# Vere Brand Reference

The canonical source is [Vere Brand Identity V1.pdf](./Vere%20Brand%20Identity%20V1.pdf), supplied on July 18, 2026.

Use it as the first reference for future Club Vere design work.

## Brand structure

- `club vere`: community events
- `vere events`: private event planning
- `vere`: parent identity

## Core palette

| Token | Value |
| --- | --- |
| Ocean blue | `#1D283A` |

| Moss green | `#37512E` |
| Olive green | `#A4C672` |
| Stone brown | `#A89C8E` |
| Chalk | `#F0EDE7` |

## Typography

- Headers: Firelli Variable Regular
- Accent: Firelli Variable Light Italic
- Body: DM Sans Regular

The PDF-embedded Firelli files are marked for preview and print embedding. They must not be republished as site webfonts. Use the approved wordmark asset for the exact brand signature and replace the temporary editorial display fallback only when licensed Firelli webfont files are provided.

## Photography

The direction is high contrast, vibrant, kinetic, and feminine.

The `vere-edit-*` images under `public/images/` were extracted from page 18. They are treated as genuine Vere event photography:

- `vere-edit-pilates.webp`
