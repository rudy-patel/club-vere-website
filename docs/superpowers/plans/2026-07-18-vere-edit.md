# The Vere Edit Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add The Vere Edit as a brand-led fifth landing-page concept using the supplied brand identity and genuine Club Vere event photography.

**Architecture:** Keep the existing shared content and design shell. Add one independent server-rendered concept component, scoped CSS, approved brand assets, and the minimal design registry changes required to expose option 5.

**Tech Stack:** Next.js 16, React 19, TypeScript 6, Tailwind CSS 4, scoped CSS, `next/image`, `next/font`, Vitest, Testing Library.

## Global Constraints

- Preserve all four existing concepts and their content.
- Commit the supplied PDF under `docs/brand/`.
- Do not redistribute the PDF-embedded Firelli files as webfonts.
- Use only the official palette and genuine page 18 event photography in option 5.
- Keep existing Instagram and Eventbrite URLs.
- No new runtime dependency, route, API, database, analytics, or fabricated event claim.
- Honor reduced motion and support 320px through wide desktop.
- Use Node 20.20.2 for verification.
